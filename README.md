# Frank's Blog

我的技术博客，记录学习心得与生活感悟。

🌐 **在线访问：** https://yufengyuanx.github.io/blog/

## 📂 内容分类

- 💻 技术笔记 - Java、Python、AI/LLM
- 📚 读书感悟 - 技术书与成长类书籍
- ✨ 生活随笔 - 工作之外的思考

## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) - Vue 驱动的静态网站生成器
- GitHub Pages - 免费托管

## 📝 本地开发

```bash
npm install
npm run docs:dev
```

访问 http://localhost:5173

## 🚀 部署

```bash
npm run docs:build
git push
```

GitHub Pages 会自动部署。
