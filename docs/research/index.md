# 🔬 Research

这一栏放的是**值得长期引用的调研与分析**，不是简单的信息收集。

---

## 什么内容会进 Research

适合进入这里的内容：
- 框架 / 工具 / 项目的深度分析
- AI engineering / workflow / agent system 方法论
- 技术方案对比与取舍
- 未来会被反复引用的结论

不适合进入这里的内容：
- 只需要快速回答的问题
- 临时调研结论
- 一次性的信息查询

这些更适合走 `research-lite`，只输出结论，不发正式文章。
